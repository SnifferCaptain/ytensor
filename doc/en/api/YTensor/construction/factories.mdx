# Factory Methods

Static methods for quick tensor creation.

## Zeros & Ones

```cpp
// All zeros
auto z = YTensor<float, 2>::zeros(3, 4);

// All ones
auto o = YTensor<float, 2>::ones(3, 4);
```

## Random Generation

```cpp
// Uniform distribution [0, 1]
auto u = YTensor<float, 2>::rand(3, 4);

// Normal distribution (Mean=0, Std=1)
auto n = YTensor<float, 2>::randn(3, 4);

// Set global random seed
YTensor<float, 2>::seed(42);
```

## Range

```cpp
// [0, 10)
auto r1 = YTensor<int, 1>::range(10);

// [5, 10)
auto r2 = YTensor<int, 1>::range(5, 10);

// [0, 10), step=2 -> 0, 2, 4, 6, 8
auto r3 = YTensor<int, 1>::range(0, 10, 2);
```

## Identity Matrix

```cpp
// 3x4 Identity (diagonal is 1, others 0)
auto I = YTensor<float, 2>::eye(3, 4);
```
