# Constructors

## Default Constructor

Creates an empty tensor, but allocates specific dimension attributes to ensure safety.

```cpp
YTensor<float, 2> t;
// t.shape() == {0, 0} ? No.
// YTensor defaults to a "scalar-like" state usually, or empty shape depending on dim.
// For static Dim > 0, shape is usually initialized to 1s or 0s. 
// Current implementation initializes shape to all 1s for robustness in shape calculation, but data is empty (nullptr).
```

## Shape Constructor

Allocates memory but **does not initialize data** (contains garbage values).

```cpp
// Create a 2x3 float tensor
YTensor<float, 2> t(2, 3);
```

## Shape + Fill Value Constructor

Allocates and fills with a value.

```cpp
// Create 2x3, fill with 1.5
YTensor<float, 2> t(2, 3, 1.5f);
```

## From `std::vector` (Data Copy)

```cpp
std::vector<float> data = {1, 2, 3, 4};
// Create 2x2 from vector
YTensor<float, 2> t({2, 2}, data);
```
