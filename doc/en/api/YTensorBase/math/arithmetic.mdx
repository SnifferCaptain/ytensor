# Arithmetic Operations (Dynamic)

`YTensorBase` supports arithmetic operations via Type Dispatch.

## Usage

Operators are overloaded to look just like static types:

```cpp
YTensorBase a = YTensorFactory::create({2, 2}, "float32");
YTensorBase b = YTensorFactory::create({2, 2}, "float32");

// Dynamic dispatch calls the correct kernel
YTensorBase c = a + b;
```

## Type Checking

If types mismatch (e.g., float32 + int32), YTensor will attempt to cast or throw a runtime error based on configuration.
Currently, mixed-type dispatch (`dispatch2`) is supported for standard numeric types.

## In-Place Operations

`+=`, `-=`, etc. are supported. Use `copy_()` for assignment-like behavior between tensors.

```cpp
// a += b
a += b;
```

## Broadcasting

Same broadcasting rules apply as `YTensor<T, D>`.

```cpp
YTensorBase m({2, 3}, "float32");
YTensorBase v({3}, "float32");

// v broadcasted to [2, 3]
auto res = m + v;
```
