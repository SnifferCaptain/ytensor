# Concat & Split

## Concat

Concatenates multiple tensors along a given dimension.

```cpp
static YTensorBase concat(const std::vector<YTensorBase>& tensors, int axis);
```

*   **Axis**: The dimension to concatenate along.
*   **Requirements**: All other dimensions must match exactly.
*   **Memory**: Allocates new memory (contiguous) for the result.
*   **Non-POD Support**: If non-POD types are registered with `copyConstruct`, `concat` supports them correctly.

**Example**:
```cpp
// shape [2, 3]
auto t1 = YTensorFactory::create({2, 3}, "float32");
auto t2 = YTensorFactory::create({2, 3}, "float32");

// Concat along dim 0 -> [4, 3]
auto res = YTensorBase::concat({t1, t2}, 0);
```

## Split

Splits a tensor into multiple view chunks along a dimension.

```cpp
std::vector<YTensorBase> split(const std::vector<int>& splitSizes, int axis) const;
```

*   **splitSizes**: List of sizes for each chunk along the axis. Sum must equal the dimension size.
*   **Memory**: Returns **Views** (shared memory).

**Example**:
```cpp
// shape [4, 3]
YTensorBase t({4, 3}, "float32");

// Split into [1, 3] + [3, 3]
auto chunks = t.split({1, 3}, 0);
```
