# Constants and Configuration

This document lists global constants and configuration items in the `yt::infos` namespace.

## Mathematical Constants

Predefined constants corresponding to types (if the type supports it):

```cpp
namespace yt::infos {
    // Epsilon (machine precision)
    constexpr double EPSILON = 1e-6; 
    
    // PI
    constexpr double PI = 3.14159265358979323846;
}
```

## Global Configuration

### BF16 Rounding Mode

This configuration affects the type conversion behavior of `bfloat16`.

```cpp
namespace yt::infos {
    // Control bfloat16 rounding mode
    // true: Round to nearest even number (Round-to-Nearest-Even) - Default behavior
    // false: Truncate (retaining high 16 bits) - Faster but less precise
    inline bool& bf16_rounding_mode() {
        static bool mode = true;
        return mode;
    }
}
```

**Usage**:

```cpp
// Set to truncate mode (global effect)
yt::infos::bf16_rounding_mode() = false;
```

### Random Seed

Used to initialize the pseudo-random number generator for `YTensor`.

*Note: Currently, different factory functions might use independent internal generators, but this constant may be used for globally resetting seeds in the future.*

*(See implementation of `YTensorFactory` for details)*
