# Backend System

This document describes the matrix multiplication backend configurations in the `yt::infos` namespace.

## MatmulBackend Enum

```cpp
enum class MatmulBackend {
    Naive = 0,  // Plain implementation, no dependencies
    Eigen = 1,  // Eigen library implementation
    AVX2 = 2    // Custom AVX2+FMA implementation
};
```

## Default Backend

```cpp
static constexpr MatmulBackend defaultMatmulBackend = 
#if YT_USE_AVX2
    MatmulBackend::AVX2
#elif YT_USE_EIGEN
    MatmulBackend::Eigen
#else
    MatmulBackend::Naive
#endif
;
```

Priority: AVX2 > Eigen > Naive

## Compilation Macros

| Macro | Description |
| --- | --- |
| `YT_USE_EIGEN` | Enables the Eigen backend |
| `YT_USE_AVX2` | Enables the AVX2 backend |
