# Backend System

## MatmulBackend Enum

Defined in `ytensor_infos.hpp`, specifies the implementation used for matrix multiplication.

```cpp
enum class MatmulBackend {
    Naive,  // Plain implementation (slow, reference)
    Eigen,  // Usually used when AVX2 is not available
    AVX2    // Highly optimized (requires hardware support)
};
```

## Global Configuration

The default backend is determined at compilation time:

```cpp
// Check the default backend
constexpr MatmulBackend defaultBackend = yt::infos::defaultMatmulBackend;
```

You can compile with macros to control this:
*   `YT_NO_AVX2`: Disable AVX2 (fallback to Eigen or Naive).
*   `YT_NO_EIGEN`: Disable Eigen (fallback to Naive).

See [Backend Selection](../guides/backend_selection.mdx) for more details.
