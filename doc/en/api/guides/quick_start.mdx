# Quick Start

This guide helps you quickly get started with the YTensor library.

## Include Header

```cpp
// Single header version (recommended for quick start)
#include "ytensor_single.hpp"

// Or modular version
#include "ytensor.hpp"
```

## Creating Tensors

### YTensor (Compile-time type and dimensions)

```cpp
// Via constructor
yt::YTensor<float, 2> a(3, 4);  // 3x4 uninitialized matrix

// Via factory methods
auto zeros = yt::YTensor<float, 2>::zeros(3, 4);   // All zeros
auto ones = yt::YTensor<float, 2>::ones(3, 4);     // All ones
auto randn = yt::YTensor<float, 2>::randn(3, 4);  // Standard normal distribution
```

### YTensorBase (Runtime type)

```cpp
yt::YTensorBase tensor = yt::YTensorBase::zeros({3, 4}, "float32");
yt::YTensorBase randn = yt::YTensorBase::randn({3, 4}, "float32");
```

## Element Access

```cpp
auto tensor = yt::YTensor<float, 2>::ones(3, 4);

// Using at()
tensor.at(0, 0) = 1.0f;
float val = tensor.at(1, 2);

// Using operator[] (YTensor only)
tensor[0][0] = 1.0f;
```

## Shape Operations

```cpp
auto tensor = yt::YTensor<float, 2>::ones(3, 4);

// Slicing (returns a view)
auto sliced = tensor.slice(0, 0, 2);  // First 2 rows

// Reshaping
auto reshaped = tensor.reshape(6, 2);

// Transposing
auto transposed = tensor.transpose();
```

## Mathematical Operations

```cpp
auto a = yt::YTensor<float, 2>::ones(3, 4);
auto b = yt::YTensor<float, 2>::ones(3, 4);

// Arithmetic operations (supports broadcasting)
auto c = a + b;
auto d = a * 2.0f;

// Matrix multiplication
auto e = a.matmul(b.transpose());
// e.shape() -> {3, 3}
```

## Full Example

```cpp
#include <iostream>
#include "ytensor_single.hpp"

int main() {

    
    // Create tensors
    auto a = yt::YTensor<float, 2>::randn(3, 4);
    auto b = yt::YTensor<float, 2>::randn(4, 5);
    
    // Matrix multiplication
    auto c = a.matmul(b);
    
    std::cout << "a shape: [" << a.shape()[0] << ", " << a.shape()[1] << "]" << std::endl;
    std::cout << "b shape: [" << b.shape()[0] << ", " << b.shape()[1] << "]" << std::endl;
    std::cout << "c shape: [" << c.shape()[0] << ", " << c.shape()[1] << "]" << std::endl;
    
    // Output tensor
    std::cout << c << std::endl;
    
    return 0;
}
```

## Compilation

```bash
g++ -std=c++20 -O3 -I/path/to/ytensor main.cpp -o main
```
