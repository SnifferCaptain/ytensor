# Activation Functions

This page documents activation functions in the `yt::function` namespace.

Normalization-related ops such as Softmax / LogSoftmax / LayerNorm are documented in [normalization.mdx](./normalization.mdx).

## Overview

Conventions:

- If a function takes `order`, `0` usually means the function value and `1` means a per-element (diagonal) derivative form.
- Unsupported `order` values throw `std::invalid_argument`.

| Function | In-place | `order` | Notes |
| --- | --- | --- | --- |
| `relu()` | `relu_()` | supports negative/0/1 (see below) | arithmetic `T` |
| `exp()` | `exp_()` | ignored | exp-compatible `T` |
| `sigmoid()` | `sigmoid_()` | `-1/0/1` | `order=-1` forwards to Softplus |
| `leakyRelu()` | `leakyRelu_()` | `0/1` | arithmetic `T` |
| `elu()` | `elu_()` | `0/1` | floating `T` |
| `selu()` | `selu_()` | `0/1` | floating `T` |
| `gelu()` | `gelu_()` | `0/1` | floating `T` |
| `tanh()` | `tanh_()` | `0/1` | floating `T` |
| `swish()` | `swish_()` | `0/1` | sigmoid-based |
| `softplus()` | `softplus_()` | `0/1` | stable softplus |
| `mish()` | `mish_()` | `0/1` | uses softplus + tanh |
| `hardSigmoid()` | `hardSigmoid_()` | `0/1` | arithmetic `T` |
| `hardSwish()` | `hardSwish_()` | `0/1` | arithmetic `T` |

## `relu()` / `relu_()`

### Core behavior

Applies ReLU element-wise.

- `order = 0`: $f(x)=\max(0, x)$
- `order = 1`: $f'(x)=1$ if $x>0$ else $0$
- `order > 1`: returns all zeros (implementation convention)
- `order < 0`: returns an integral-form variant for $x>0$ (see implementation details)

### Signature

```cpp
template<typename T, int dim>
YTensor<T, dim> relu(const YTensor<T, dim>& x, int order = 0);

template<typename T, int dim>
YTensor<T, dim>& relu_(YTensor<T, dim>& x, int order = 0);
```

### Parameters

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | input tensor |
| `order` | `int` | `0` | order selector |

### Returns

- out-of-place: `YTensor<T, dim>`
- in-place: `YTensor<T, dim>&`

### Example

```cpp
#include "ytensor_function.hpp"

auto x = yt::YTensor<float, 1>({5}, {-2, -1, 0, 1, 2});
auto y = yt::function::relu(x);
yt::function::relu_(x);
```

## `sigmoid()` / `sigmoid_()`

### Core behavior

Applies a numerically stable sigmoid element-wise.

- `order = 0`: $\sigma(x)=\frac{1}{1+e^{-x}}$
- `order = 1`: $\sigma(x)(1-\sigma(x))$
- `order = -1`: forwards to Softplus ($\log(1+e^x)$)

### Signature

```cpp
template<typename T, int dim>
YTensor<T, dim> sigmoid(const YTensor<T, dim>& x, int order = 0);

template<typename T, int dim>
YTensor<T, dim>& sigmoid_(YTensor<T, dim>& x, int order = 0);
```

### Parameters

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | input tensor |
| `order` | `int` | `0` | supports `-1/0/1` |

### Returns

- out-of-place: `YTensor<T, dim>`
- in-place: `YTensor<T, dim>&`

### Example

```cpp
auto x = yt::YTensor<float, 1>({3}, {-1.0f, 0.0f, 1.0f});
auto y0 = yt::function::sigmoid(x, 0);
auto y1 = yt::function::sigmoid(x, 1);
auto sp = yt::function::sigmoid(x, -1);
```

## `leakyRelu()` / `leakyRelu_()`

### Core behavior

Leaky ReLU:

- `order = 0`: $\max(x, \alpha x)$
- `order = 1`: derivative form (`1` for $x>0$, else `alpha`)

### Signature

```cpp
template<typename T, int dim>
YTensor<T, dim> leakyRelu(const YTensor<T, dim>& x, T alpha = static_cast<T>(0.01), int order = 0);

template<typename T, int dim>
YTensor<T, dim>& leakyRelu_(YTensor<T, dim>& x, T alpha = static_cast<T>(0.01), int order = 0);
```

### Parameters

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | input tensor |
| `alpha` | `T` | `0.01` | slope for the negative side |
| `order` | `int` | `0` | supports `0/1` |

### Returns

- out-of-place: `YTensor<T, dim>`
- in-place: `YTensor<T, dim>&`

### Example

```cpp
auto x = yt::YTensor<float, 1>({4}, {-2, -1, 1, 2});
auto y = yt::function::leakyRelu(x, 0.1f);
```

## `elu()` / `elu_()`

### Core behavior

ELU:

- `order = 0`: $x>0?x:\alpha(e^x-1)$
- `order = 1`: $x>0?1:\alpha e^x$

### Signature
