# Activation Functions

Common activation functions in the `yt::functional` or global namespace.

## ReLU

$$ \text{ReLU}(x) = \max(0, x) $$

```cpp
auto y = tensor.relu();
// or
yt::functional::relu(tensor);
```

## Sigmoid

$$ \text{Sigmoid}(x) = \frac{1}{1 + e^{-x}} $$

```cpp
auto y = tensor.sigmoid();
```

## Tanh

$$ \text{Tanh}(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}} $$

```cpp
auto y = tensor.tanh();
```

## Softmax

Computes softmax along a specified dimension.

$$ \text{Softmax}(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}} $$

```cpp
// Along the last dimension
auto y = tensor.softmax(-1);
```

**Note**: Softmax is typically numerically stable (implements max-subtraction trick).

## GELU

Gaussian Error Linear Unit.

```cpp
auto y = tensor.gelu();
```
