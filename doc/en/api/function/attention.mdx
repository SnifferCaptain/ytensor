# Attention

Implements standard Scaled Dot-Product Attention.

$$ \text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right)V $$

## API

```cpp
YTensorBase attention(
    const YTensorBase& q, 
    const YTensorBase& k, 
    const YTensorBase& v, 
    const std::optional<YTensorBase>& mask = std::nullopt
);
```

**Parameters**:

*   `q`: Query tensor, shape `[batch, num_heads, seq_len, head_dim]`
*   `k`: Key tensor, shape `[batch, num_heads, seq_len, head_dim]`
*   `v`: Value tensor, shape `[batch, num_heads, seq_len, head_dim]`
*   `mask`: Optional mask (usually causal mask or padding mask).

**Returns**:

*   Result tensor of shape `[batch, num_heads, seq_len, head_dim]`.

## Flash Attention

*Experimental Support*

If enabled and hardware supports it, `attention` calls may automatically dispatch to FlashAttention kernels for speedup and memory saving.
