# Pooling

This page documents 1D pooling functions in the `yt::function` namespace.

## Overview

- Negative axes are normalized by: `axis = (axis % dim + dim) % dim`.
- If `stride < 0`, the implementation uses `stride = kernelSize`.

| Function | In-place | Notes |
| --- | --- | --- |
| `maxPool1d()` | - | output size follows `kernelSize` / `stride` |
| `avgPool1d()` | - | output size follows `kernelSize` / `stride` |

---

## `maxPool1d()`

### Core behavior

Performs 1D max pooling along `axis`.

- Output size: `output_size = (input_size - kernelSize) / stride + 1`.
- Throws `std::invalid_argument` if `output_size <= 0`.

### Signature

```cpp
template<typename T, int dim>
YTensor<T, dim> maxPool1d(const YTensor<T, dim>& x, int kernelSize, int stride = -1, int axis = -1);
```

### Parameters

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` | - | input tensor |
| `kernelSize` | `int` | - | window size |
| `stride` | `int` | `-1` | step; `<0` means `kernelSize` |
| `axis` | `int` | `-1` | pooling axis |

### Returns

- `YTensor<T, dim>` with the pooled axis resized to `output_size`

### Example

```cpp
#include "ytensor_function.hpp"

auto x = yt::YTensor<float, 1>({8}, {1, 3, 2, 0, 5, 1, 2, 4});
auto y = yt::function::maxPool1d(x, 2, 2, -1);
```

## `avgPool1d()`

### Core behavior

Performs 1D average pooling along `axis`.

### Signature

```cpp
template<typename T, int dim>
YTensor<T, dim> avgPool1d(const YTensor<T, dim>& x, int kernelSize, int stride = -1, int axis = -1);
```

### Parameters

| Name | Type | Default | Description |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` | - | input tensor |
| `kernelSize` | `int` | - | window size |
| `stride` | `int` | `-1` | step; `<0` means `kernelSize` |
| `axis` | `int` | `-1` | pooling axis |

### Returns

- `YTensor<T, dim>` with the pooled axis resized to `output_size`

### Example

```cpp
auto x = yt::YTensor<float, 1>({8}, {1, 3, 2, 0, 5, 1, 2, 4});
auto y = yt::function::avgPool1d(x, 2, 2, -1);
```
