# Compression

YTensor uses **Zlib** for data compression.

## Strategy

When saving a tensor, you can select whether to compress:

```cpp
io.save(tensor, "name", true);  // Enable compression (Default)
io.save(tensor, "name", false); // Disable compression
```

### Compression Implementation

*   **Only Payload Data is compressed**.
*   The Tensor Header and File Index are always uncompressed plain text/binary for quick scanning.
*   Compression is performed per-tensor chunk.

### Performance Trade-off

| Feature | `compress=true` | `compress=false` |
| --- | --- | --- |
| **Space** | 30% - 90% (depends on sparsity) | 100% (raw data) |
| **Write Speed** | Slower (CPU bound) | Fastest (IO bound) |
| **Read Speed** | Slower (Decompression overhead) | Fastest (Direct read) |

**Recommendation**:
*   For disk storage: **Enable** (space is expensive).
*   For IPC / Temp files: **Disable** (latency is critical).
