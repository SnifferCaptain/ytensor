# YTensor 中文文档

> 现代 C++20 轻量级多维张量库 —— header-only，极简集成，科研/竞赛/工程皆宜

## 欢迎使用 YTensor

YTensor 是一个纯头文件的 C++ 张量库，专为需要高性能多维数组操作的场景设计。它提供了类似 NumPy/PyTorch 的接口，但无需任何运行时依赖，只需包含一个头文件即可使用。

## 特性亮点

- 🧩 **Header-only**：只需 `ytensor_single.hpp`，零第三方依赖，直接 `#include` 即用
- 📐 **任意维度**：支持 shape、切片、转置、reshape、permute 等常用操作
- ⚡ **多功能**：支持并行、广播、基础数学、少量常用的深度学习函数（matmul/softmax/attention）
- 🛠️ **易扩展**：源码清晰，便于二次开发
- 🗂️ **I/O 支持**：可保存/加载 YTensor 支持的文件格式
- 🔢 **广泛的类型支持**：支持常用的数据类型，如 float16、bfloat16 等

## 快速开始

### 安装

查看 [安装指南](./installation/installation.mdx) 了解如何将 YTensor 集成到您的项目中。

### 基本示例

```cpp
#include "ytensor_single.hpp"

int main() {
    // 创建一个 3x4 的浮点型张量
    yt::YTensor<float, 2> a(3, 4);
    
    // 创建全1张量
    auto b = yt::YTensor<float, 2>::ones(3, 4);
    
    // 访问和修改元素
    a.at(1, 2) = 5.0f;
    
    // 执行运算
    auto c = a + b;
    
    // 输出张量
    std::cout << c << std::endl;
    
    return 0;
}
```
