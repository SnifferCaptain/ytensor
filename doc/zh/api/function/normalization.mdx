# 归一化

本文档介绍 `yt::function` 命名空间中的归一化相关函数（Softmax / LogSoftmax / LayerNorm）。

## 概述

说明约定：

- `axis` 支持负数索引；实现会标准化为：`axis = (axis % dim + dim) % dim`。
- 下文的 `order = 1` 均为“逐元素/对角线形式”的导数结果（不是完整 Jacobian，也不包含与上游梯度的链式组合）。

| 函数 | 说明 | 原地版本 | `order` 支持 | 类型约束 |
| --- | --- | --- | --- | --- |
| `softmax()` | Softmax 归一化 | `softmax_()` | - | 需要 `std::exp` |
| `logSoftmax()` | LogSoftmax：`x - logsumexp(x)` | `logSoftmax_()` | `0/1` | `T` 为浮点 |
| `layerNorm()` | LayerNorm：沿 `axis` 归一化 | `layerNorm_()` | `0/1` | `T` 为浮点 |

---

## `softmax()` / `softmax_()`

### 核心功能描述

沿指定轴执行 Softmax：

$$\mathrm{softmax}(x_i) = \frac{e^{x_i}}{\sum_j e^{x_j}}$$

实现采用数值稳定版本：对每个 slice 先减去最大值 `max`，再计算 `exp(x - max)` 并归一化。

- 返回张量形状与输入一致。
- 对连续张量路径做了优化（类似“融合 max/exp/sum”的循环结构）；非连续张量走通用 `at()` 访问路径。

### 函数签名

```cpp
template<typename T, int dim>
YTensor<T, dim> softmax(const YTensor<T, dim>& x, int axis = -1);

template<typename T, int dim>
YTensor<T, dim>& softmax_(YTensor<T, dim>& x, int axis = -1);
```

### 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | 输入张量 |
| `axis` | `int` | `-1` | 归一化轴（支持负数索引） |

### 返回值

- 非原地：`YTensor<T, dim>`
- 原地：`YTensor<T, dim>&`

### 使用示例

```cpp
#include "ytensor_function.hpp"

void example_softmax() {
    auto x = yt::YTensor<float, 2>({2, 3}, {1, 2, 3, 1, 2, 3});

    auto y = yt::function::softmax(x, -1);
    yt::function::softmax_(x, -1);
}
```

## `logSoftmax()` / `logSoftmax_()`

### 核心功能描述

沿指定轴计算 LogSoftmax：

$$\mathrm{logSoftmax}(x) = x - \log\sum\exp(x)$$

- `order = 0`：返回 `logSoftmax` 本身（内部使用 `max` + `log(sum(exp(x-max)))` 的稳定方式）。
- `order = 1`：实现返回对角线形式：$1 - \mathrm{softmax}(x)$。

注意：严格意义上，LogSoftmax 的雅可比矩阵并非对角矩阵；这里的 `order = 1` 是一种简化输出，适合作为某些推导/验证用途。

### 函数签名

```cpp
template<typename T, int dim>
YTensor<T, dim> logSoftmax(const YTensor<T, dim>& x, int axis = -1, int order = 0);

template<typename T, int dim>
YTensor<T, dim>& logSoftmax_(YTensor<T, dim>& x, int axis = -1, int order = 0);
```

### 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | 输入张量 |
| `axis` | `int` | `-1` | 计算轴（支持负数索引） |
| `order` | `int` | `0` | 支持 `0/1`；其他值会抛异常 |

### 返回值

- 非原地：`YTensor<T, dim>`
- 原地：`YTensor<T, dim>&`

### 使用示例

```cpp
auto x = yt::YTensor<float, 2>({2, 3}, {1, 2, 3, 1, 2, 3});

auto y = yt::function::logSoftmax(x, -1, 0);
auto diag = yt::function::logSoftmax(x, -1, 1);

yt::function::logSoftmax_(x, -1, 0);
```

## `layerNorm()` / `layerNorm_()`

### 核心功能描述

沿指定轴进行 Layer Normalization（无可学习的 affine 参数）：

$$y = \frac{x - \mu}{\sqrt{\sigma^2 + \varepsilon}}$$

其中 $\mu$ 和 $\sigma^2$ 在 `axis` 维度上计算。

- `order = 0`：返回归一化结果。
- `order = 1`：实现返回对角线近似（或称简化版）导数：

$$\frac{\partial y_i}{\partial x_i} = \frac{1}{\sqrt{\sigma^2+\varepsilon}}\left(1 - \frac{1}{N} - \frac{\hat{x}_i^2}{N}\right)$$

其中 $N$ 是 `axis` 维度长度、$\hat{x}$ 为归一化后的值。

### 函数签名

```cpp
template<typename T, int dim>
YTensor<T, dim> layerNorm(const YTensor<T, dim>& x, int axis = -1, T eps = static_cast<T>(1e-5), int order = 0);

template<typename T, int dim>
YTensor<T, dim>& layerNorm_(YTensor<T, dim>& x, int axis = -1, T eps = static_cast<T>(1e-5), int order = 0);
```

### 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | 输入张量 |
| `axis` | `int` | `-1` | 归一化轴（支持负数索引） |
| `eps` | `T` | `1e-5` | 数值稳定项，避免除 0 |
| `order` | `int` | `0` | 支持 `0/1` |

### 返回值

- 非原地：`YTensor<T, dim>`
- 原地：`YTensor<T, dim>&`

### 使用示例

```cpp
auto x = yt::YTensor<float, 2>::randn({2, 4});

auto y = yt::function::layerNorm(x, -1, 1e-5f, 0);
auto diag = yt::function::layerNorm(x, -1, 1e-5f, 1);

yt::function::layerNorm_(x, -1);
```
