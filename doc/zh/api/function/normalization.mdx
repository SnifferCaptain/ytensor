# 归一化

本文档介绍 `yt::function` 命名空间中的归一化相关函数。

## 概述

说明约定：

- 负轴会在内部标准化。
- 这些归一化算子不再提供简化版 `order` 导数接口。
- 大多数函数同时支持单轴重载与 `std::vector<int>` 多轴重载。

| 函数 | 说明 | 原地版本 | 仿射参数 | 多轴支持 |
| --- | --- | --- | --- | --- |
| `layerNorm()` | 零均值、单位方差归一化 | `layerNorm_()` | 可选 `gamma` / `bias` | 支持 |
| `rmsNorm()` | RMS 归一化 | `rmsNorm_()` | `weight` | 支持 |
| `normalize()` | Lp 范数归一化 | `normalize_()` | 可选 `gamma` / `bias` | 支持 |
| `instanceNorm()` | 每样本每通道的空间归一化 | `instanceNorm_()` | 可选 `gamma` / `bias` | 由 `channelAxis` 推导 |
| `batchNorm()` | 朴素 BatchNorm | `batchNorm_()` | 可选 `gamma` / `bias` | 由 `channelAxis` 推导 |
| `groupNorm()` | GroupNorm | `groupNorm_()` | 可选 `gamma` / `bias` | 由 `channelAxis` 与 `groups` 推导 |

---

## `layerNorm()` / `layerNorm_()`

### 核心功能描述

沿单个轴或多个轴执行零均值、单位方差归一化：

$$y = \frac{x - \mu}{\sqrt{\sigma^2 + \varepsilon}}$$

### 函数签名

```cpp
template<typename T, int dim>
YTensor<T, dim> layerNorm(const YTensor<T, dim>& x, const std::vector<int>& axes, T eps = static_cast<T>(1e-5));

template<typename T, int dim>
YTensor<T, dim> layerNorm(const YTensor<T, dim>& x, int axis = -1, T eps = static_cast<T>(1e-5));

template<typename T, int dim>
YTensor<T, dim>& layerNorm_(YTensor<T, dim>& x, const std::vector<int>& axes, T eps = static_cast<T>(1e-5));

template<typename T, int dim>
YTensor<T, dim>& layerNorm_(YTensor<T, dim>& x, int axis = -1, T eps = static_cast<T>(1e-5));
```

### 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | 输入张量 |
| `axes` / `axis` | `std::vector<int>` / `int` | - / `-1` | 归一化轴 |
| `eps` | `T` | `1e-5` | 数值稳定项 |

### 返回值

- 支持带 `gamma` / `bias` 的仿射重载。
- 输出形状与输入一致。

### 使用示例

```cpp
#include "ytensor_function.hpp"

void example_softmax() {
    auto x = yt::YTensor<float, 3>::randn({2, 3, 4});
    auto y = yt::function::layerNorm(x, std::vector<int>{1, 2});
    yt::function::layerNorm_(x, -1);
}
```

## `rmsNorm()` / `rmsNorm_()`

### 核心功能描述

沿单个轴或多个轴执行 RMS 归一化：

$$y = \frac{x}{\sqrt{\mathrm{mean}(x^2) + \varepsilon}} \odot weight$$

### 函数签名

```cpp
template<typename T, int dim, int weightDim>
YTensor<T, dim> rmsNorm(const YTensor<T, dim>& x, const YTensor<T, weightDim>& weight, const std::vector<int>& axes, T eps = static_cast<T>(1e-5));

template<typename T, int dim, int weightDim>
YTensor<T, dim> rmsNorm(const YTensor<T, dim>& x, const YTensor<T, weightDim>& weight, int axis = -1, T eps = static_cast<T>(1e-5));
```

### 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | 输入张量 |
| `weight` | `const YTensor<T, weightDim>&` | - | 与归一化轴匹配的缩放参数 |
| `axes` / `axis` | `std::vector<int>` / `int` | - / `-1` | 归一化轴 |
| `eps` | `T` | `1e-5` | 数值稳定项 |

### 返回值

- 输出形状与输入一致。

### 使用示例

```cpp
auto x = yt::YTensor<float, 2>::randn({2, 8});
auto w = yt::YTensor<float, 1>::ones({8});
auto y = yt::function::rmsNorm(x, w, -1);
```

## `normalize()` / `normalize_()`

### 核心功能描述

沿单个轴或多个轴执行 Lp 范数归一化：

$$y = \frac{x}{\max(\lVert x \rVert_p, \varepsilon)}$$

### 函数签名

```cpp
template<typename T, int dim>
YTensor<T, dim> normalize(const YTensor<T, dim>& x, const std::vector<int>& axes, T p = static_cast<T>(2), T eps = static_cast<T>(1e-12));

template<typename T, int dim>
YTensor<T, dim> normalize(const YTensor<T, dim>& x, int axis = -1, T p = static_cast<T>(2), T eps = static_cast<T>(1e-12));
```

### 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` / `YTensor<T, dim>&` | - | 输入张量 |
| `axes` / `axis` | `std::vector<int>` / `int` | - / `-1` | 归一化轴 |
| `p` | `T` | `2` | Lp 范数阶数 |
| `eps` | `T` | `1e-12` | 分母下界 |

### 返回值

- 支持带 `gamma` / `bias` 的仿射重载。
- 输出形状与输入一致。

### 使用示例

```cpp
auto x = yt::YTensor<float, 3>::randn({2, 3, 4});
auto y = yt::function::normalize(x, std::vector<int>{1, 2}, 2.0f);
```

## `instanceNorm()` / `batchNorm()` / `groupNorm()`

这三类函数都建立在统一的多轴归一化后端之上：

- `instanceNorm()`：对每个样本、每个通道的空间维做归一化。
- `batchNorm()`：按通道统计 batch 维与空间维，是朴素实现，不包含 EMA 与推理融合。
- `groupNorm()`：先按组划分通道，再对组内通道和空间维联合归一化。

它们都提供可选的 `gamma` / `bias` 仿射重载，输出形状与输入一致，并通过 `channelAxis` 推导实际归一化轴。
