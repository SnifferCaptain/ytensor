# 池化

本文档介绍 `yt::function` 命名空间中的 1D 池化函数。

## 概述

说明约定：

- `axis` 支持负数索引；实现会标准化为：`axis = (axis % dim + dim) % dim`。
- `stride < 0` 时，默认取 `stride = kernelSize`。

| 函数 | 说明 | 原地版本 | 类型约束 |
| --- | --- | --- | --- |
| `maxPool1d()` | 最大池化 | - | `T` 为算术类型 |
| `avgPool1d()` | 平均池化 | - | `T` 为算术类型 |

---

## `maxPool1d()`

### 核心功能描述

沿 `axis` 执行 1D 最大池化。

- 输出长度：`output_size = (input_size - kernelSize) / stride + 1`。
- 若 `output_size <= 0`，会抛出 `std::invalid_argument`（通常是 `kernelSize` 过大）。

### 函数签名

```cpp
template<typename T, int dim>
YTensor<T, dim> maxPool1d(const YTensor<T, dim>& x, int kernelSize, int stride = -1, int axis = -1);
```

### 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` | - | 输入张量 |
| `kernelSize` | `int` | - | 窗口大小 |
| `stride` | `int` | `-1` | 步幅；`<0` 时默认等于 `kernelSize` |
| `axis` | `int` | `-1` | 池化轴（支持负数索引） |

### 返回值

- `YTensor<T, dim>`
- shape 与输入相同，但 `axis` 维度会被替换为 `output_size`

### 使用示例

```cpp
#include "ytensor_function.hpp"

void example_maxpool1d() {
    auto x = yt::YTensor<float, 1>({8}, {1, 3, 2, 0, 5, 1, 2, 4});

    auto y = yt::function::maxPool1d(x, /*kernelSize=*/2, /*stride=*/2, /*axis=*/-1);
}
```

## `avgPool1d()`

### 核心功能描述

沿 `axis` 执行 1D 平均池化。

- 输出长度：`output_size = (input_size - kernelSize) / stride + 1`。

### 函数签名

```cpp
template<typename T, int dim>
YTensor<T, dim> avgPool1d(const YTensor<T, dim>& x, int kernelSize, int stride = -1, int axis = -1);
```

### 参数说明

| 参数名 | 类型 | 默认值 | 说明 |
| --- | --- | --- | --- |
| `x` | `const YTensor<T, dim>&` | - | 输入张量 |
| `kernelSize` | `int` | - | 窗口大小 |
| `stride` | `int` | `-1` | 步幅；`<0` 时默认等于 `kernelSize` |
| `axis` | `int` | `-1` | 池化轴（支持负数索引） |

### 返回值

- `YTensor<T, dim>`
- shape 与输入相同，但 `axis` 维度会被替换为 `output_size`

### 使用示例

```cpp
auto x = yt::YTensor<float, 1>({8}, {1, 3, 2, 0, 5, 1, 2, 4});

auto y = yt::function::avgPool1d(x, 2, 2, -1);
```
